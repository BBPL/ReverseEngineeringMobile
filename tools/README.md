# Tools Guide:

## Dex2Jar
Linux/macOS:
```
$ . dex2jar.sh [*.apk]
```

Windows:
```
$ ./dex2jar.bat [*.apk]
```

## Decompiling

### JD-GUI
```
$ java -jar jd-gui.jar [*.jar]
```

[Source](http://java-decompiler.github.io/)

### Procyon
```
$ java -jar decompiler.jar -jar [*.jar]
```
[Source](https://bitbucket.org/mstrobel/procyon/wiki/Java Decompiler)
